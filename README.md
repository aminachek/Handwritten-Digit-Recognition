# Handwritten Digit Recognition 🔢

Application permettant de reconnaître des chiffres écrits à la main à l’aide d’un modèle de machine learning, avec une interface simple et intuitive pour tester la reconnaissance en temps réel.

## 🔧 Technologies

Python, NumPy, TensorFlow / Keras, HTML, CSS, JavaScript (interface simple)

## ✨ Fonctionnalités

* Reconnaissance des chiffres manuscrits (0 à 9)
* Interface utilisateur simple pour dessiner un chiffre
* Prédiction automatique via un modèle entraîné
* Affichage du résultat en temps réel

## 📌 Description

Ce projet utilise des techniques de machine learning pour analyser et reconnaître des chiffres manuscrits. L’utilisateur peut dessiner un chiffre dans une interface graphique, puis le système prédit automatiquement le chiffre correspondant.

## 🚧 État du projet

Version fonctionnelle avec améliorations possibles (optimisation du modèle et design de l’interface).
